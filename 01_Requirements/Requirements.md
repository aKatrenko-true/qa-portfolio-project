# Требования к системе

| ID | Требование | Приоритет |
|----|-----------|-----------|
| FR-01 | Пользователь может зарегистрироваться с валидными данными | High |
| FR-02 | Email должен проходить форматную валидацию | High |
| FR-03 | Пароль должен содержать не менее 8 символов | High |
| FR-04 | При неверных данных отображается сообщение об ошибке | Medium |
| FR-05 | Пользователь может авторизоваться с валидными данными | High |
| FR-06 | При неверном пароле доступ не предоставляется | High |
| FR-07 | Пользователь может выйти из системы | Medium |
| FR-08 | API возвращает корректные HTTP-статусы | Medium |
| FR-09 | Данные пользователя корректно сохраняются в базе | High |
| FR-10 | Интерфейс корректно отображается на мобильных устройствах | Low |
