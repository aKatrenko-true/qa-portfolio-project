# Функциональные требования

1. Пользователь может войти в систему
2. Пользователь может выйти из системы
3. Пользователь может просматривать список товаров
4. Пользователь может открыть карточку товара
5. Пользователь может добавить товар в корзину
6. Пользователь может удалить товар из корзины
7. Пользователь может оформить заказ
8. Обязательные поля валидируются
9. После оформления заказа отображается подтверждение
