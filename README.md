# QA Портфолио Проект  
Ручное тестирование веб-приложения Demoblaze (интернет-магазин)

---

## 📌 Описание проекта

Данный проект демонстрирует полный цикл ручного тестирования веб-приложения (e-commerce).

Цель проекта — показать практическое применение QA-процесса:

- Анализ требований
- Составление тест-плана
- Разработка тест-кейсов
- Проведение smoke-тестирования
- Функциональное тестирование
- API-тестирование
- Оформление баг-репортов
- Подготовка итогового отчёта

---

## 🛠 Объект тестирования

Приложение: https://www.demoblaze.com  
Тип: веб-приложение (интернет-магазин)

Протестированные модули:
- Регистрация пользователя
- Авторизация
- Каталог товаров
- Корзина
- Оформление заказа
- Форма "Contact"
- Навигация

Не входило в тестирование:
- Нагрузочное тестирование
- Тестирование безопасности
- Автоматизация
