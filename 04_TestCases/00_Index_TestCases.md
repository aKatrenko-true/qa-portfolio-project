# Test Cases — Index

## Описание
Набор тест-кейсов для веб-приложения. Тест-кейсы разделены по слоям и функциональным областям (UI / API / SQL / Mobile),
чтобы отражать рабочий подход к тестированию.

## Содержание
1. **01_UI_Registration_TestCases.md** — регистрация пользователя (позитив/негатив/валидации/устойчивость)
2. **02_UI_Login_Session_TestCases.md** — авторизация и пользовательская сессия + общие UI проверки
3. **03_UI_Access_Control_TestCases.md** — контроль доступа (защищённые страницы, поведение после logout)
4. **04_API_TestCases.md** — базовые API проверки (статусы, ошибки, токены)
5. **05_SQL_TestCases.md** — проверки данных в БД (создание пользователя, уникальность, хранение пароля)
6. **06_Mobile_TestCases.md** — базовые проверки мобильного отображения и кликабельности

## Формат тест-кейса
**ID · Название · Priority · Preconditions · Steps · Expected Result**
