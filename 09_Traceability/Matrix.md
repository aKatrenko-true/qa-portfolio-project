# Матрица трассируемости требований

| ID требования | Описание | ID тест-кейса | Статус |
|--------------|----------|--------------|--------|
| FR-01 | Пользователь может зарегистрироваться | TC-REG-01 | Passed |
| FR-02 | Пользователь может войти в систему | TC-LOG-01 | Passed |
| FR-03 | Добавление товара в корзину | TC-CART-01 | Passed |
| FR-04 | Удаление товара из корзины | TC-CART-02 | Passed |
| FR-05 | Оформление заказа | TC-CHK-01 | Failed |
| FR-06 | Отправка формы Contact | TC-CON-01 | Failed |
